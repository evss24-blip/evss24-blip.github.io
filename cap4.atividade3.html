Roteiro de estudos

Para manter nossos estudos em dia e sempre conseguirmos estudar, devemos criar um roteiro de estudos.

  1. Estudar
    1. Ler resumos.
    2. Assistir video aulas.
    3. Escrever resumos
  2. Praticar.
    1. Fazer exercicios.
    2. corrigir os exercicios e ser tiver errada, fazer ate acertar
  3. Não mexer no celular.
    1. É muito importante não mexer no celular enquanto estuda, pois distrai facilmente.
    2. Libera dopamina, ou seja, quando mexer vai ser dificil voltar a estudar.

Acima de tudo devemos priorizar nossa saúde mental, nossas notas não vão estar boa se não estivermos bem.

  1. Separar algumas horas do seu dia.
    1. Não adianta estudar por 8h se no outro dia não lembrar de nada, encontre duas horas livres em sua rotina pra estudar.
  2. Encontre professores bons.
    1. tente encontrar professores que você entende facilmente.
  3. Saiba de qual maneira você aprende mais rápido.
    1. sempre temos uma maneira de aprender mais fácil, seja escrevendo, lendo ou ouvindo.

5 tecnologias web

Vamos listar em ordem decrescente 5 tecnologias web, que é tudo que é usado para criar coisas na internet

  1. Banco de Dados (MySQL, PostgreSQL)

    é um sistema usado para armazenar e organizar informações

  2. Git / GitHub

    Git é um sistema de controle de versões, ja GitHubum é um site/plataforma que usa o Git.

  3. JavaScript

    é a linguagem que faz o site funcionar e reagir

  4. Css

    é uma linguagem para deixar o site com uma aparência melhor.

  5. Html

    é uma linguagem usada para criar a estrutura de sites
